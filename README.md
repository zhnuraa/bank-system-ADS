# bank-system-ADS

Nurassyl Zhumagul, IT - 2501.


Description:
My program simulates a mini banking system. I used LinkedList to store bank accounts dynamically, Stack to store transaction history using LIFO, and Queue to manage bill payments and account requests using FIFO. I also used an array to demonstrate physical data storage with fixed size. All these structures are integrated through Bank, ATM, and Admin menus.
